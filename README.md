# Ativa Real Fit Virtual

## Project Documentation
This project aims to provide a virtual fitness experience, allowing users to engage in various workouts and track their fitness goals.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/RogerioPonciano/ativa-real-fit-virtuall.git
   cd ativa-real-fit-virtuall
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```

## Customization Guide
To customize the application:
1. Modify the configuration files located in the `/config` directory.
2. Update the user interface as per your preferences in the `/src/components` directory.
3. Add new workout routines in the `/data/workouts.json` file.

For further assistance, refer to the documentation files in the `/docs` directory.
